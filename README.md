# GeoEscape

[![License: AGPL v3](https://img.shields.io/badge/License-AGPL%20v3-blue.svg)](https://www.gnu.org/licenses/agpl-3.0)

## Descifra la realidad

GeoEscape es un juego de escape room basado en GPS. Crea, comparte y juega casos interactivos con amigos.

**Web:** [https://escape.manumontaraz.es](https://escape.manumontaraz.es)
**Código fuente:** [https://github.com/ManuMontaraz/geoescape](https://github.com/ManuMontaraz/geoescape)

## Características

- 🗺️ **Juego basado en GPS**: Navega por el mundo real para resolver puzles y escapar
- 🛠️ **Editor de casos**: Crea tus propias aventuras con un editor visual intuitivo
- ☁️ **Casos en la nube**: Sube y comparte casos con la comunidad
- 📱 **Offline-first**: Funciona sin conexión una vez descargado el caso
- 🎮 **Demo incluida**: Prueba el caso demo sin necesidad de registro
- 🗳️ **Sistema de votos**: Valora los casos de otros creadores

## Tecnologías

- **Frontend**: HTML5 Canvas, Leaflet (mapas), Font Awesome
- **Backend**: PHP, MySQL
- **Offline**: Service Worker, IndexedDB
- **Geolocalización**: GPS del navegador (solo en cliente)
